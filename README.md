# Fancy Arenas

A Minecraft 1.21 plugin for creating custom D&D-inspired, story-based mob arenas.

## Features

- **Wave-based Combat** - Design multi-wave arena battles
- **Cutscenes & Dialogs** - Create immersive story-driven experiences
- **Diverse Entities** - Custom mobs that can swap items and use unique abilities
- **Shops** - Integrated shop system for players

## Requirements

- Minecraft 1.21
- Bukkit/Spigot/Paper server

## Installation

1. Download the plugin JAR
2. Place it in your server's `plugins` folder
3. Restart the server
4. Configure arenas in the generated config files

## Commands

| Command | Alias | Description | Permission |
|---------|-------|-------------|------------|
| `/moditem [options]` | - | Modify item attributes and metadata | `FancyArenas.moditem` |
| `/fancymob [options]` | - | Create custom mobs | `FancyArenas.fancymob` |
| `/fancyitem [options]` | - | Manage custom items | `FancyArenas.fancyitem` |
| `/fancyarena [options]` | `/fa` | Set up custom arenas and dungeons | `FancyArenas.fancyarena` |
| `/shop` | - | Open arena shop | `FancyArenas.shop` |
| `/cbscan <radius>` | - | Scan for command blocks | `FancyArenas.cbscan` |
| `/rm <selector>` | - | Remove an entity | `FancyArenas.rm` |
